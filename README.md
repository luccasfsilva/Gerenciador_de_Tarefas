# Task Manager CLI - Java

Projeto simples de gerenciamento de tarefas feito em Java para praticar Programação Orientada a Objetos.

## Funcionalidades

- Adicionar tarefas
- Listar tarefas
- Marcar tarefas como concluídas
- Remover tarefas

## Tecnologias

Java

## Conceitos aplicados

- Programação Orientada a Objetos
- ArrayList
- Entrada de dados com Scanner
- Organização de código em múltiplas classes

## Como executar

Compilar:

javac *.java

Rodar:

java Main
